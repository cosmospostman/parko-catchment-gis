# V10

Joint S1+S2 TAM. Part of a three-way comparison (S2-only / S1-only / joint S1+S2).

**Current best: CVaR25=0.845, macro AUC=0.940** (d_ff=1024, batch_size=4096, 2026-05-26)

---

## Feature set

16 bands: B02 B03 B04 B05 B07 B08 B8A B11 B12 + NDVI + NDWI + MAVI + NDRE + CI_RE (14 S2, identical to V9) + s1_vh + s1_vv (2 S1).  
B06 and EVI excluded. Derived S1 cols (vh_vv ratio, RVI) deferred until discriminative value confirmed in sweep.  
Pixel z-score normalisation. Band stats computed per-source (S2 and S1 rows normalised independently).

---

## Training sites

Sites: Norman Road, Cloncurry (absence only), Etna Creek (train split), Landsend, Lake Mueller, Corfield, Roper River, Hughenden, Burdekin, Maria Downs, Rupert Creek, Frenchs.  
Val holdout: Etna Creek (presence + absence), Landsend, Hughenden, Burdekin, Maria Downs, Rupert Creek, Frenchs.

Identical to V9-SPECTRAL. Frenchs included to address bare-soil confusion and maintain monsoonal site diversity.

## Training history

### Initial — V10 created as joint S1+S2 experiment (2026-05-21)

V10 forked from V9-SPECTRAL at the point where S1 pipeline wiring was complete (mixed-mode `TAMDataset`, per-source DOY normalisation, `is_s1` mask propagated through the Transformer pooling step). V9 frozen as the S2-only comparator.

S1 feature set restricted to raw VH and VV backscatter (`s1_vh`, `s1_vv`) — total 16 input bands. Derived columns (`s1_vh_vv`, `s1_rvi`) excluded until a sweep confirms their value over the raw backscatter pair. Architecture scaled up from V9 (d_model 128→256, n_layers 2→3) to match the increased input dimensionality.

### d_ff sweep — 2026-05-26

`d_ff` was never set in `model_kwargs`, falling through to the `TAMConfig` default of 64 — a carry-over from when `d_model=32`. At d_model=256 this is a 0.25× ratio. Fixed by setting `d_ff=512` explicitly in the experiment. Sweep results (all other params identical, 60 epochs):

| d_ff | CVaR25 | macro AUC | notes |
|------|--------|-----------|-------|
| d_ff | CVaR25 | macro AUC | batch_size | notes |
|------|--------|-----------|------------|-------|
| 64   | 0.785  | 0.910     | 8192       | implicit default (baseline) |
| 512  | 0.794  | 0.918     | 8192       | — |
| 1024 | 0.845  | 0.940     | 4096       | **new default**; early stop epoch 53 (best epoch 38) |

d_ff=1024 is a decisive step change (+0.060 CVaR, +0.030 macro over 512). The halved batch size to avoid OOM is not a concern given the result. d_ff=512 was not the ceiling. d_ff=1024 + batch_size=4096 adopted as the experiment default.
